Installing Games via Tinfoil

Heads up: Tinfoil is unreliable. We recommend DBI on PC instead — it's faster, more stable, and doesn't depend on third-party servers that go down. We're keeping this guide because some people insist on trying Tinfoil anyway. If that's you, manage expectations.

What Tinfoil is

Tinfoil is a Switch app that installs NSP files. Unlike DBI, it doesn't read files from your PC or USB — it pulls them from servers over Wi-Fi. Convenience is the appeal: no PC needed, install over Wi-Fi.

The catch: those servers are run by random groups, go offline often, and there's nothing anyone can do when they fail. The free server "Libera" used to be reliable but is long gone. Most "shared" servers shared in jailbreak groups have similar fates. When Tinfoil works, it's nice. When it doesn't, you're stuck.

App Mode vs Applet Mode

Tinfoil needs to run in App Mode to work properly. App Mode gives it more memory and system privileges.

  • App Mode (correct): highlight a game on the home screen, hold R, then launch — pick HBMenu, then Tinfoil.
  • Applet Mode (wrong): opening HBMenu via the Album button. Memory is too limited, Tinfoil will crash or fail.

Time and date sync

Tinfoil needs the correct system time or it refuses to connect. Set it up once:

  1. Connect your Switch to Wi-Fi
  2. System Settings → set timezone to UTC+8 (Philippines time)
  3. Enable Network Time
  4. Open DBI → Tools → NTP Time Sync

Search function

Tinfoil's search lets you find games on your connected servers. Same caveat as everything else here: only works when the server is up.
