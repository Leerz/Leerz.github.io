Fix Joy-Cons That Won't Pair

TL;DR: Start with Level 1 (basic resync). 90% of pairing issues fix here. Only escalate if it doesn't work.

Joy-Con pairing problems are annoying but almost always solvable. We've ranked the fixes from simplest to most involved — start at the top and work down only if needed.

Level 1: Basic resync

Solves most pairing issues. Takes 2 minutes.

  1. In Jailbreak mode, attach the Joy-Cons to the Switch (handheld orientation)
  2. Go to Settings → Controllers and Sensors
  3. Scroll down and choose Disconnect Controllers (hold X)
  4. Detach the Joy-Cons
  5. Reattach them
  6. Detach again and disconnect them in settings once more
  7. Reboot into Legit Mode and repeat the process

Test if they pair properly now. If yes, you're done. If no, continue to Level 2.

Level 2: Advanced pairing with data sync from OFW

This pulls clean pairing data from the official firmware and writes it to your jailbroken side. Takes about 10 minutes.

Heads up: involves running a script in Tegraexplorer. Don't skip steps.

  1. Boot into Legit Mode (OFW), Joy-Cons attached
  2. Settings → Disconnect Controllers
  3. Detach Joy-Cons. Press and hold the sync button until SL/SR lights turn off
  4. Disable Bluetooth (or enable airplane mode)
  5. Power off the Switch
  6. Power on in Legit Mode
  7. Attach Joy-Cons
  8. Re-enable Bluetooth (disable airplane mode)
  9. Repeat steps 3, 4, 5, and 6 once more
  10. Power off the Switch
  11. Boot into Hekate
  12. Open Payloads
  13. Launch Tegraexplorer
  14. Run the script syncfix.te
    Note: this triggers a red initialization screen — that's normal, your data is safe
  15. Wait for the script to finish, then power off
  16. Boot into Legit Mode (OFW) and check if Joy-Cons pair

If still broken, escalate to Level 3.

Level 3: Joy-Con Factory Mode method

Last resort. Forces the Joy-Cons into factory/transport mode and rebuilds the pairing from zero.

This involves accessing the Joy-Con's internal recovery state — it's safe if you follow the video carefully, but there's no margin for skipping steps.

  3. Reboot the Switch

  5. Reattach and confirm they sync properly

Still won't pair?

If all three levels fail, the Joy-Con itself may have a hardware fault (Bluetooth chip, antenna, or controller board). Bring it in — we can diagnose and repair.

Different problem: Joy-Cons drift?

Drift is a different issue (analog stick wear, not pairing). See the FAQ on Joy-Con drift.
