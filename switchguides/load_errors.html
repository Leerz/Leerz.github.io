Fix Games That Won't Load

TL;DR: 9 times out of 10 it's missing files (incomplete install, missing sigpatches) or a save data mismatch. Start with #1 below.

Games crashing or refusing to launch is one of the most common things we get asked about — and almost always fixable. Don't panic. Work through the list below in order; the first cause is by far the most common.

1. Missing or incomplete game data

Every game has up to four parts:

  • Base game — the core data. Required.
  • Update — patches and fixes. Optional but common.
  • DLC — extra content. Optional.
  • Ticket — authorization data. Sometimes required, especially for newer games.

If a game won't load, the most likely culprit is one of these:

  • Install was interrupted (low storage, disconnect, etc.)
  • You installed an update or DLC without the base game
  • The ticket file is missing

Fix: Reinstall the game, making sure you have the base game file (the biggest one). If you have all the parts and it still fails, try installing them in order — base → update → DLC.

2. Save data mismatch

This one trips a lot of people up. Save data is tied to a specific game version. If those versions don't line up, the game refuses to load (or crashes when you press Continue).

Common scenarios:

  • You updated the game and the save conversion failed
  • You rolled back to an older version after playing on a newer one

Especially common with Zelda, Pokémon, and Animal Crossing.

Fix options:

  1. Back up your save first with JKSV or DBI (see Save Management)
  2. Try matching the game version to the save (re-install the version you played on)
  3. If desperate: delete the save and start fresh
  4. Last resort: use DBI's tools to clean up orphaned game data, then reinstall

3. Missing sigpatches

Sigpatches are what let your modded Switch run unsigned games (your backups). If they're missing or outdated for your CFW version, nothing will launch.

If every game crashes — not just one — this is almost certainly your problem.

Fix: Update your CFW pack from PremazingPH (which includes matching sigpatches). See Updating Firmware Safely.

4. Fake SD card

Counterfeit SD cards claim huge capacities but actually have a fraction of the storage. They appear to install games normally — until you try to load them and the data is corrupted.

Fix: Test your SD card with H2testw (Windows) or F3 (Mac/Linux). If it's fake, replace it with a genuine SanDisk. See the FAQ entry on fake SD cards for details.

5. Hardware faults

Rare but real. Failing RAM, a degrading APU, or a bad SD card reader can all cause game crashes.

If you've tried everything above and games still won't load — especially if this used to work fine and suddenly stopped — message us. Hardware diagnosis isn't something you can do at home, but we can.
