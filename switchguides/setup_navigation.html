General Setup & Navigation

Switching between Legit Mode (OFW) and Jailbreak Mode (CFW) on your Nintendo Switch is essential for managing games and apps safely. Below is a guide to help you navigate through both modes:

1. Switching Between Legit Mode and Jailbreak Mode

In Legit Mode (OFW), you have access to online features like the eShop. Jailbreak Mode (CFW) is used for homebrew apps, mods, and backups. To switch, use the bootloader menu (Hekate) to select which mode to boot into.

2. Navigating Hekate Bootloader

Hekate is the key to controlling your Switch's boot options. It allows you to boot into Legit Mode, Jailbreak Mode, or even Android/Linux if set up. You can also access tools like backup, restore, and custom configurations.
