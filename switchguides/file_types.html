Understanding File Types

When managing games on your jailbreak Switch, it's important to understand the different file types you’ll encounter. These file types are used to store base games, updates, downloadable content (DLC), and other essential game data. Below, we explain the most common file formats you'll come across and their uses:

Base Game Files

Base game files typically come in the following formats:

  • XCI – This format is often used for cartridge dumps of the game.
  • NSP – These are usually eShop dumps or copies of the game.
  • NSZ – A compressed version of NSP files, smaller in size but identical in content.

Example:

  • Hades v0 XCI – A cartridge dump of the base game.
  • Hades v0 NSP – The base game in eShop dump format.
  • Hades v0 NSZ – The base game in a compressed NSP format.

Updates and DLC Files

Game updates and DLC files are typically available in either NSP or NSZ formats:

  • Hades v1346xxx NSP/NSZ – This is usually the game’s update file in either NSP or compressed NSZ format.
  • Hades Weapon Pack DLC v0 NSP/NSZ – This is a downloadable content (DLC) file, usually smaller than the base game files.

Super XCI (One File for Everything)

Some XCI files come bundled with the base game, update, and DLC in a single package. This is what we call a Super XCI, and it’s a convenient all-in-one file:

  • Hades v0 + v1346 + DLC XCI – A Super XCI file that contains the base game, update, and DLC all in one XCI package. It can be installed using DBI or any compatible installer.

File Archives (ZIP, RAR)

When downloading game files online, they are often compressed into ZIP or RAR files. These compressed archives will contain NSP, XCI, or NSZ files, and need to be extracted before use:

  • Download the entire ZIP or RAR archive.
  • If the archive is split into multiple parts, make sure to download all parts.
  • Extract the first file in the archive to access the contents, including NSP, XCI, or NSZ files.
