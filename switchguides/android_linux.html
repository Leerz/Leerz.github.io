Setting Up Linux or Android on a Working Switch

Adding Linux or Android as additional boot options on a jailbroken Nintendo Switch is a great way to expand its functionality. However, there are several important considerations to keep in mind before proceeding.

Key Considerations

  • Version Compatibility: Not all versions of Android are compatible with every Switch model:
    • Android 10/11 works on Erista (v1 unpatched models).
    • Android 12 and newer should work on all models, but each version has its quirks.
  • SD Card Space: These extra boot options take up significant space on the SD card. Ensure you have enough storage available.
  • Learning Curve: Linux and Android have their own user interfaces and functionalities, requiring some familiarity to use effectively.
  • Barebones Preinstallation: Our preinstallations of Linux and Android are kept barebones to provide a "fresh from factory" experience, allowing you to customize as needed.
